Stack

¿Qué es un Stack?

Un stack es una pila o conjunto de tecnologías y herramientas que se utilizan para desarrollar una aplicación, son todas las tecnologías que usa el frontend y el backend

Ejemplos de Stack

    LAMP

  • LINUX como sistema operativo en el servidor
  • APACHE como servidor web
  • MySQL como base de datos
  • PHP como lenguaje de programación

    MEAN

  • MONGO como base de datos
  • EXPRESS como framework de nodejs
  • ANGULARJS para la parte del frontend
  • NODEJS para la parte del backend

    MERN

  • MONGO como base de datos
  • EXPRESS como framework de nodejs
  • REACTJS para la parte del frontend
  • NODEJS para la parte del backend

    EDteam

  • LINUX en los servidores
  • PostgreSQL como base de datos
  • GO como lenguaje de backend
  • ECHO como framework
  • REACTJS para el frontend
  • EDgrid para los estilos
